# Кратко об 1S моделях BETAFPV полетников 

## [Matrix 1S Brushless Flight Controller (3IN1)](https://betafpv.com/collections/brushless-flight-controller/products/matrix-1s-brushless-flight-controller-hd)

Используется под цифровые видеосистемы

## [Matrix 1S Brushless Flight Controller (4IN1)](https://betafpv.com/collections/brushless-flight-controller/products/matrix-1s-brushless-flight-controller-hd)  
Крест без VTX но уже есть аналоговый OSD.  
ELRS приемник на борту.

## [Matrix 1S Brushless Flight Controller (5IN1)](https://betafpv.com/collections/brushless-flight-controller/products/matrix-1s-brushless-flight-controller)  
Так называемый, ромб. Уже все есть.

## [Air Brushless Flight Controller (4IN1)](https://betafpv.com/collections/brushless-flight-controller/products/air-brushless-flight-controller)  
Ромб без одного угла, нет ELRS. Максимально облегчен для гонщиков. Нет коннекторов моторов. Тонкий текстолит, от этого чаще гнется или ломается при крашах.

## [Air Brushless Flight Controller (5IN1)](https://betafpv.com/collections/brushless-flight-controller/products/air-brushless-flight-controller)  
Ромб все в одном. Максимально облегчен для гонщиков. Нет коннекторов моторов. Тонкий текстолит, от этого чаще гнется или ломается при крашах.
