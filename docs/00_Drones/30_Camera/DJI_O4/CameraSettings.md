# Настройки камеры

## Для Gyroflow

Чтобы DJI O4 (как и в случае с O3 / Avata 2) корректно записывал данные гироскопа в метаданные видеофайла, нужно строго соблюдать комбинацию настроек камеры. Если ошибиться хоть в одном пункте, гироскоп просто не запишется.

- Стабилизация: OFF - Встроенную стабилизацию (RockSteady / HorizonSteady) нужно полностью отключить.
Угол обзора (FOV): Широкий (Wide) - Режимы Ультраширокий (Ultra Wide), Нормальный (Normal) или Естественный (Natural Wide) не записывают данные гироскопа.
  
- Пропорции кадра (Aspect Ratio): желательно 4:3 - пишется максимум несжатой информации с матрицы по вертикали. Потом в Gyroflow можно сделать кроп под 16:9. На 16:9 гироскоп тоже может писаться, но запас для стабилизации «желеек» и углов будет сильно меньше.

**Опциональные настройки:**

- Resolution: 4K/60fps или 4K/30fps

- Выдержка (Shutter) (FPS*2): 120 для 60fps или 60 для 30fps  

- Режим: Manual  

- ISO: как можно ниже (100). Если есть возможность - использовать ND-фильтры  

- EV: -0.3  - чтобы уменьшит пересветы

- Баланс белого: ручной режим. около 5600K для солнечной погоды и 6000–6500K для вечера и пасмурной



## ND Filter

Номинал фильтра (ND8, ND16, ND32 и т.д.) показывает, во сколько раз он уменьшает количество поступающего света.

| Фильтр | Сколько света пропускает | Когда использовать (При ISO 100 и Выдержке 1/120) |
| --- | --- | --- |
| **Без фильтра** | 100% | Смеркается, сильная облачность, полет в густом лесу. |
| **ND8** | 12.5% | Переменная облачность, «золотой час» (закат/рассвет). |
| **ND16** | 6.25% | Обычный дневной свет, легкая облачность. |
| **ND32** | 3.12% | Яркий, безоблачный солнечный день. |
| **ND64** | 1.56% | Ослепительное солнце, полеты над снегом или водой летом. |
